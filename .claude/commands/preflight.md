Run a preflight check before building.

You must inspect the project before editing.

Follow this exactly:

1. Read:
   - CLAUDE.md
   - CONTRACTS.md
   - LESSONS.md
   - package scripts
   - relevant source files

2. Identify:
   - existing architecture
   - protected contracts
   - likely files to change
   - likely tests/checks to run
   - risks before editing

3. Produce a short plan:
   - what will change
   - what will not change
   - what must not break
   - how success will be verified

4. Do not edit code during preflight unless the user explicitly asks.
